# CloudBoost Homebrew Tap

Homebrew tap for installing CloudBoost on macOS.

## Install

```bash
brew tap victorbrandaao/cloudboost
brew install --cask cloudboost
```

## Update

```bash
brew update
brew upgrade --cask cloudboost
```

## Uninstall

```bash
brew uninstall --cask cloudboost
```

CloudBoost website: <https://victorbrandaao.github.io/CloudBoost/>

Latest release: <https://github.com/victorbrandaao/CloudBoost/releases/latest>
